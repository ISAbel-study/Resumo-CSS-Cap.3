Resumo Capítulo 3 - Estilizando Páginas Web usando CSS

Introdução e Princípios do CSS

CSS é uma liguagem de estilo utilizada para controlar como os elementos HTML são visualizados no frontend

Estrutura de uma Regra CSS:

  • Selector: Identifica quais elementos HTML serão afetados pela regra.
  • Declaração: A declaração define o estilo propiamente dito, um bloco de declarações contém uma ou mais declarações separadas por ponto-e-vírgula e vários blocos são delimitados por chaves. Cada declaração inclui o nome de uma propriedade CSS(ex:cor, fonte, tamanho) e um valor(ex:azul, Arial, 12), separados por dois pontos.

Seletores e Especificidade

Os seletores são usados para definir quais elementos HTML são afetados por uma regra CSS

  • Tipos de seletores:seletores de tipo (tags como h1, p), existem seletores de classe (.classe), ID seletor(#id), seletor universal(*), seletor de atributo, e seletores de hierarquia (descendentes, filhos diretos, irmãos).
  • Pseudo-classes e Pseudo-elementos:Usados para estilizar estados específicos (ex: <:hover> para passar o mouse) ou partes de um elemento (ex: <::first-line>).
  • Especificidade: Quando múltiplas regras conflitam, o navegador calcula a especificidade baseada em contadores: (A) IDs, (B) classes/atributos/pseudo-classes e (C) tipos/pseudo-elementos. Se os seletores forem os mesmos, a regra CSS do último seletor definido é usada e caso eles sejam diferentes o seletor mais específico é usado.
  • Herança: Muitas propriedades (como fontes e cores) são herdadas dos elementos pais para os elementos filhos, a menos que uma regra mais específica seja definida.

Formatação de Tudo

Formatação de Texto e Cores
Texto e Cores O CSS permite definir cores de texto (<color>) e fundo (<background-color>), além de famílias de fontes (<font-family>), tamanhos (<font-size>), estilo da fonte(<font-weight> ou <font-style>), alinhamentos (<text-align>) e espaçamentos entre linhas (<line-height>).
Listas É possível personalizar marcadores de listas com <list-style-type> para especificar sua aparência, dependendo do tipo da lista diferentes valores são aplicáveis para essa propriedade. Usar imagens personalizadas com <list-style-image> e controlar o recuo do texto na lista com <list-style-position> (valores inside ou outside).
Tabelas O design de tabelas envolve propriedades como <border> para as bordas de toda a tabela, <border-collapse> para unir bordas próximas e <padding> para definir o espaçamento interno das células. O uso de pseudo-classes como <:nth-child>(even/odd) é comum para criar linhas zebradas.

Sistemas de Layout e Posicionamento

Isso serve para definir a posição de um elemento HTML em uma página web

Princípios de Posicionamento:
  • Block-level vs. Inline-level: Elementos de bloco começam em uma nova linha e ocupam toda a largura disponível; elementos inline ocupam apenas o espaço necessário e ficam lado a lado.
  • Tipos de Posicionamento: A propriedade <position> define como um elemento é posicionado na página: <static> :fluxo normal, <relative> :relativo à sua posição no ‘fluxo normal’, <absolute> : posição em relação ao elemento pai posicionado, <fixed> : relativo à janela do navegador e <sticky> : ao rolar a página até um ponto especificado o elemento é fixado.
Layouts Modernos
  • Float Layout: Pode ser usado para determinar o comportamento do elementos em relação ao texto, com a propriedade <float>. No entanto hoje existem duas alternativas melhores, layout flexbox e o layout grid.
  • Flexbox Layout (Unidimensional): Foca no ajuste dinâmico de espaço e alinhamento de itens em uma única direção (linha ou coluna). Um 'container flex' ajusta dinamicamente o tamanho de seus 'itens flex' para preencher o espaço disponível ou evitar transbordamento tanto para arranjos verticais quanto para horizontais.
  • Grid Layout (Bidimensional): Permite o posicionamento flexível de elementos em uma grade de linhas e colunas, facilitando a criação de layouts complexos e estruturados. Define-se um 'grid container' e as áreas onde os 'grid items' devem se posicionar.
